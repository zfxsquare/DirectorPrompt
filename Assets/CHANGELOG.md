# 更新日志

* 修复了知识检索配置未生效的问题
* 为项目编辑界面新增通用提示文本, 以帮助确定哪些字段会否进入/不进入生成检索流程
* 修正了知识系统
    * 知识条目“标题” → “备注”
    * 知识条目“标签” → “关键词”
* 简化了状态栏更新逻辑, 现在仅发送 AI 确实可以修改的条目, 避免分散注意力
* 极大简化了配置读取逻辑
* 移除了状态值“复合”类型
* 使部分遗漏的枚举也进入了本地化体系
* 重构了状态系统中“枚举”类型的配置方式
    * 移除了与该类型不相关的界面元素
    * 可以单独指定每个枚举如何触发, 包含“随机”与“表达式两种方式”, “随机”方式下可以为枚举单独指定概率,
      “表达式”方式下可以为该枚举关联其他“数值”类型的状态属性, 并在满足表达式时切换至该枚举
* 调整软件图标为无背景版本
* 统一将多处位于底部的“添加”按钮挪动到内容区右上角，与标题同行
